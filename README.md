# 🇧🇷 Político — RPG de Vida Política Brasileira

Jogo de simulação de carreira política brasileira completa.

## Como jogar

- Crie seu personagem com nome, personalidade e partido
- Comece como **Vereador** e suba na carreira
- Enfrente **50 crises por mandato** com escolhas reais
- Gerencie o **orçamento** por área (saúde, educação, segurança...)
- Proponha leis, troque de partido, aceite ou recuse subornos
- Chegue à **Presidência da República** ou se aposente com legado

## Cargos disponíveis

🏙️ Vereador · 🏛️ Prefeito · 🏟️ Dep. Estadual · 🎯 Dep. Federal · ⭐ Senador · 🗺️ Governador · 🎖️ Presidente

## Rodar localmente

```bash
npm install
npm run dev
```
